## Currency Converter üí∞

A real-time currency converter that fetches the latest exchange rates from an API.

### Features:

* **Live Exchange Rates**: Uses the [Currency API](https://github.com/fawazahmed0/currency-api) to get up-to-date currency conversion rates.
* **Dynamic Flag Updates**: The country flag changes as you select a different currency.
* **User-Friendly Interface**: Simple and intuitive design for easy use.

### How to Use:

1.  Enter the amount you want to convert.
2.  Select the "From" and "To" currencies from the dropdown menus.
3.  Click the "Get Exchange Rate" button to see the result.

A classic Rock Paper Scissors game where you can play against the computer.

### Features:

* **Interactive Gameplay**: Click on your choice of rock, paper, or scissors to play.

2.  The computer will make a random choice.
3.  The winner is announced, and the scores are updated.

## Tic Tac Toe Game ‚ùå‚≠ï

A simple and fun implementation of the classic Tic Tac Toe game.

### Features:

* **Two-Player Gameplay**: Play against a friend on the same device.
* **Win and Draw Detection**: The game automatically detects when a player has won or when the game is a draw.
* **Reset and New Game**: Easily start a new game at any time.

### How to Play:

1.  Player O starts the game.
2.  Click on an empty box to place your mark.
3.  The first player to get three of their marks in a row (horizontally, vertically, or diagonally) wins.

A weather dashboard that provides real-time weather information for any location.

### Features:

* **Real-Time Weather Data**: Fetches current weather and a 5-day forecast from the [OpenWeatherMap API](https://openweathermap.org/api).
* **Search Functionality**: Search for any city or location to get its weather details.
* **Detailed Information**: Displays temperature, "feels like" temperature, weather description, and more.

### How to Use:

2.  Click the search button or press Enter.
3.  The current weather and a 5-day forecast will be displayed.

## Calculator üßÆ

A basic calculator with standard arithmetic operations.

### Features:

* **Standard Operations**: Perform addition, subtraction, multiplication, and division.
* **Clear and Delete**: Includes buttons to clear the entire input or delete the last character.
* **Responsive Design**: The calculator is designed to work well on different screen sizes.

### How to Use:

1.  Click the number and operator buttons to create an expression.
2.  Click the equals (`=`) button to see the result.
3.  Use the "AC" button to clear all input or "DE" to delete the last entry.
